FAQ

Frequently Asked Questions

Quick answers to common questions

How does VirtualAI understand my commands?

VirtualAI uses pattern matching to understand natural language commands. Simply type what you want to do, like "create file notes.txt" or "send email to john@example.com", and the assistant will interpret and execute your request.

Where are my files stored?

All files are stored securely on the server in a designated uploads directory. The file metadata is also stored in a MySQL database for quick access and searching.

Can I recover deleted files?

Currently, deleted files cannot be recovered. The assistant always asks for confirmation before deleting to prevent accidental deletions.

What email service does VirtualAI use?

VirtualAI uses PHP's mail function by default. For production use, you can configure it to use SMTP services like Gmail, SendGrid, or Mailgun for better deliverability.
